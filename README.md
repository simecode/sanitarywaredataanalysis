# 📊 卫浴与泛家居进出口数据洞察智库 (Sanitary Ware Data Observer)

这是一个专为卫浴及泛家居行业打造的**自动化数据清洗、可视化与 AI 洞察报告生成工具**。
本工具能够将繁杂的中国海关进出口原始数据（如 6910 卫生陶瓷、848180 水暖五金等），一键转化为具有可视化的深度市场观察报告。

## ✨ 核心功能 (Features)

* **免代码自动化清洗**：上传海关原始 Excel 数据后，系统自动完成注册地解析、贸易伙伴区域映射、异常值清理及 Yo Y（同比）计算。
* **多维度全景统计**：支持“年度全景全周期统计”与“前 N 个月动态监测”双模式灵活切换。
* **市场格局可视化**：内置 Plotly 交互式图表，直观呈现前十大出口目的地榜单及高增长潜力市场的气泡分布矩阵。
* **AI 深度撰稿引擎**：需要自主填入API 密钥使用。
* **一键下载底稿**：自动导出包含多 Sheet 维度的结构化 Excel 底层数据，方便二次加工。

## 🚀 部署与使用 (Usage)

本项目原生支持在 [Streamlit Community Cloud](https://share.streamlit.io/) 上进行零成本一键部署。

### 1. 环境依赖
详见 `requirements.txt`：
* `streamlit`
* `pandas`
* `openpyxl`
* `plotly`
* `requests`

### 2. 初始化配置
* **区域映射表**：请确保仓库根目录中包含名为 `区域映射表.xlsx` 的文件，系统将在后台自动读取该文件为全球市场进行大区归类。
* **字段映射配置** ：上传文件后会自动出现一个配置区，包含7个字段的下拉选择器，系统会先智能猜测并预填，用户只需确认或修改。

### 3. 运行流程
1. 打开部署好的网页端。
2. 在左侧边栏选择**分析维度**。
3. 拖拽上传单个月份或跨年度的多个**海关原始 Excel 数据包**。
4. 点击 **"🚀 开始数据解码与报告生成"**，等待图表渲染与 AI 报告产出。
